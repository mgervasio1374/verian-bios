Verian BIOS Architecture

This system is a multi-tenant Business Intelligence Operating System designed for:
- Sales Intelligence
- Marketing Automation
- Workflow Orchestration
- AI-assisted decisioning

Core requirements:
- Multi-tenancy (tenant_id enforced)
- Workflow runtime (Inngest)
- Supabase Postgres + RLS
- Artifact/document system
- Messaging layer with Resend
- Config-driven architecture

Claude must follow Phase 1–2 architecture before implementing advanced features.
